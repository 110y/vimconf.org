VimConf 2016

  • 開催日: 2016/11/05 (Sat)
  • 場所: mixi株式会社 @渋谷ファーストタワー7階
  • 時間: 13:00 (発表者の数次第では 10:00 になる可能性あり) ～ 18:30 & 懇親会 18:30～
  • 状況: 発表者募集中
  • Date: 2016/11/05 (Sat)
  • Place: mixi at Shibuya First Tower 7th floor.
  • Time: 13:00 (or 10:00, depending number of speakers) - 18:30 & convival party 18:30-
  • Status: Call for Presentations

来たる 2016年11月5日(土) に、今年も Vim カンファレンス (VimConf 2016) を開催いたします。

現在 VimConf 2016 では発表者を募集しています。 テーマは「Vim もしくはテキスト編集に関わることならなんでも」です。 1つの発表の長さは各30分程度で、募集枠は 8 枠です。 発表の応募はこちらの GitHub issue からお願いいたします。

また同時にライトニングトーク (LT) の発表者も募集しています。 こちらは制限時間 5分 で、募集枠は 5 枠です。 LTの応募はこちらの GitHub issue からお願いいたします。

発表者には交通費の一部が支給されるほか、VimConf 2016 の参加費の免除を予定しています。

募集締め切りは8月末日です。 応募多数の場合は、枠数を調整したり、公平でありつつも独断的な審査において選定させていただくことを、予めご了承ください。

We will present a conference for Vim, a.k.a. VimConf 2016 at 5th Nov, 2016 in Tokyo, Japan.

The call for speakers is now open. We are looking for 8 regular talks (about 30 minutes length). Theme of presentation is "anything related with Vim or text editing." Please check GitHub issue to register for regular talks.

And we are looking 5 lightening talks 5 minutes length too. Please check GitHub issue to register for lightening talks.

Registration will be closed end of Aug, 2016.
